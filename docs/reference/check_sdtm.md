# Check SDTM domains DM, EX and PC for missing date and time information.

Check SDTM domains DM, EX and PC for missing date and time information.

## Usage

``` r
check_sdtm(sdtm, verbose = TRUE)
```

## Arguments

- sdtm:

  The SDTM as SDTM object.

- verbose:

  Boolean to indicate whether to include details.

## Value

Nothing.

## Examples

``` r
check_sdtm(examplinib_poc)
```
