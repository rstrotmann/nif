# Add baseline lean body mass (LBM)

Add baseline lean body mass (LBM)

## Usage

``` r
add_bl_lbm(obj, method = lbm_boer)
```

## Arguments

- obj:

  A nif object.

- method:

  The function to calculate LBM, i.e., [`lbm_boer()`](lbm_boer.md),
  [`lbm_hume()`](lbm_hume.md) or [`lbm_peters()`](lbm_peters.md).

## Value

A nif object.
