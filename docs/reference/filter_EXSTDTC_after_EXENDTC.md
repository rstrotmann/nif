# Remove administrations with EXSTDTC after EXENDTC

Remove administrations with EXSTDTC after EXENDTC

## Usage

``` r
filter_EXSTDTC_after_EXENDTC(ex, dm, silent = NULL)
```

## Arguments

- ex:

  The ex domain as data frame.

- dm:

  The dm domain as data frame.

- silent:

  Suppress messages, defaults to nif_option setting, if NULL.

## Value

A data frame.
