# Get the name of a function

Returns the name of a function that is supplied as an argument.

## Usage

``` r
function_name(fun)
```

## Arguments

- fun:

  A function.

## Value

The name of the function as a character string.

## Examples

``` r
function_name(mean)
#> [1] "mean"
function_name(sum)
#> [1] "sum"
```
