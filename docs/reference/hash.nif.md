# Generate the XXH128 hash of a nif object

Generate the XXH128 hash of a nif object

## Usage

``` r
# S3 method for class 'nif'
hash(obj)
```

## Arguments

- obj:

  A nif object.

## Value

The XXH128 hash of the nif object as character.

## Examples

``` r
hash(examplinib_sad_nif)
#> [1] "7c2013d7f0ad6ade972c1dca52879eb5"
```
