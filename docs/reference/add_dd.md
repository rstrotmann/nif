# Add Field definition to Data Definition Table

Add Field definition to Data Definition Table

## Usage

``` r
add_dd(
  obj,
  name,
  definition,
  type,
  description,
  unit = NA_character_,
  source = ""
)
```

## Arguments

- obj:

  Data definition Table as data frame.

- name:

  Field name as character.

- definition:

  Field definition as character.

- type:

  Field type as character.

- description:

  Field description as character.

- unit:

  Field unit as character.

- source:

  Field source as character.

## Value

A data frame.
