# Trial title as reported in TS domain

Trial title as reported in TS domain

## Usage

``` r
trial_title(obj)
```

## Arguments

- obj:

  A sdtm object.

## Value

The title or NULL.

## Examples

``` r
trial_title(examplinib_sad)
#> [1] "An open label dose escalation study of RS2023 in healthy subjects"
```
