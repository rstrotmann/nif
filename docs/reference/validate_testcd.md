# Validate that one or multiple testcd are included in a sdtm object

Validate that one or multiple testcd are included in a sdtm object

## Usage

``` r
validate_testcd(sdtm, testcd, domain = NULL)
```

## Arguments

- sdtm:

  A sdtm object.

- testcd:

  Testcode to validate as character.

- domain:

  Domain as character

## Value

Validated testcode(s) as character.
