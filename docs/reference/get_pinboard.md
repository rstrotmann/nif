# Get pinboard object

Get pinboard object

## Usage

``` r
get_pinboard(board = NULL)
```

## Arguments

- board:

  The path to the board object. Defaults to the 'pinboard' value of
  nif_option, or the value of the 'NIF_PINBOARD' key in the .Renviron
  setting if the 'nif_option' is not set.

## Value

A pinboard object.
