# Generate the XXH128 hash of a sdtm object

Generate the XXH128 hash of a sdtm object

## Usage

``` r
# S3 method for class 'sdtm'
hash(obj)
```

## Arguments

- obj:

  A sdtm object.

## Value

The XXH128 hash of the sdtm object as character.

## Examples

``` r
hash(examplinib_sad)
#> [1] "de028ea936140b7e5b45bd294f296923"
```
