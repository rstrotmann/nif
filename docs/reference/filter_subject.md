# Keep only selected USUBJID in the data set

Keep only selected USUBJID in the data set

## Usage

``` r
filter_subject(obj, usubjid)
```

## Arguments

- obj:

  The input, either a `nif` or `sdtm` object.

- usubjid:

  The USUBJID as character.

## Value

The filtered object.
