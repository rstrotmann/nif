# Apply CAT and SCAT filter

Apply CAT and SCAT filter

## Usage

``` r
apply_cat_filter(obj, param, param_field)
```

## Arguments

- obj:

  A data frame.

- param:

  The parameter (e.g., the cat or scat value).

- param_field:

  The field in the data frame (e.g., PCCAT, PCSCAT).

## Value

A data frame.
