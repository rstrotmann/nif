# nif class constructor

nif class constructor

## Usage

``` r
new_nif(obj = NULL, ..., silent = NULL)
```

## Arguments

- obj:

  A data frame containing the actual NIF data or a sdtm object.

- ...:

  Further arguments.

- silent:

  Suppress messages, as logical. Defaults to nif_option setting if NULL.

## Value

A nif object from the input data set.
