# nif: Build and Explore NONMEM Input Format Data Sets from Clinical Study Data

A System to Convert Study Data Tabulation Model (SDTM) Formatted Data
into Longitudinal Data Tables Suitable for Nonlinear Mixed Effects
Modeling (NONMEM). Aggregate Drug Administrations, PK/PD Observations
and Covariates in a Stepwise Manner. Explore, Summarize and Visualize
SDTM and NONMEM Data Sets.
