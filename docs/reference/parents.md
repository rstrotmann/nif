# Parent compounds within a NIF object

Parent compounds within a NIF object

## Usage

``` r
parents(obj)
```

## Arguments

- obj:

  A NIF object.

## Value

The parent compounds as character.

## Examples

``` r
parents(examplinib_poc_nif)
#> [1] "RS2023"
parents(examplinib_poc_min_nif)
#> [1] "1"
```
