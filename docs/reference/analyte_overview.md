# Analyte overview

Overview on the analytes included in a nif object and their respective
parent treatments

## Usage

``` r
analyte_overview(obj)
```

## Arguments

- obj:

  A nif object.

## Value

A data frame.

## Examples

``` r
analyte_overview(examplinib_poc_nif)
#>      ANALYTE PARENT
#> 1     RS2023 RS2023
#> 2 RS2023487A RS2023
```
